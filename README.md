# HealthMandala — Frontend

A smart doctor appointment booking system built with React.

## Tech Stack
- React 19 + React Router v7
- Firebase (auth)
- CSS Modules

## Getting Started

```bash
npm install
npm start
```

## Features
- Patient & Doctor auth with sliding panel UI
- Token-based appointment booking
- Progressive date display
- Doctor schedule management with AM/PM picker
- Digital prescriptions
- Admin supervision dashboard
