Problem Statement
What Problem
Does It Solve?
😤

Long Queues

Patients wait hours at hospitals with no idea of their position or wait time.

📋

No Doctor Info

Finding the right specialist, availability, and fees requires multiple visits.

📅

Unmanaged Scheduling

Doctors have no digital tool to manage their schedule, leading to overbooking.

📄

Paper Prescriptions

Handwritten prescriptions are easily lost. No digital medical history exists.

⚕️
Healthcare in Nepal

🏙️ Urban Hospitals

Overcrowded OPDs with 3–5 hour average wait times for consultation

📵 No Digital System

Most clinics still rely on paper tokens and manual appointment books

🔍 Lack of Transparency

Patients cannot verify doctor credentials or check real-time availability

Our Solution
How HealthMandala
Solves It
🎫

Token-Based Booking

Auto-calculated tokens from working hours. 30 min/patient. No overbooking.

🔍

Find the Right Doctor

Browse by specialty, view credentials, fees, hospital, and live availability.

💊

Digital Prescriptions

Doctors issue digital prescriptions. Patients access full history anytime.

🛡️

Admin Verification

NID, NMC certificate, and degree documents reviewed before doctors go live.

✅
Key Innovations

📅 Progressive Date Display

Shows only the nearest available date. Next date unlocks when current is fully booked.

📱 OTP Verification

Twilio SMS OTP ensures only real users can register on the platform.

🕐 AM/PM Schedule Picker

Doctors set working hours with intuitive AM/PM time pickers — no 24h confusion.

Development Methodology
Agile Development
Approach
🔄

Iterative Sprints

Built in short cycles — each sprint delivered a working, testable feature increment.

🧩

Component-Based

Frontend split into reusable React components. Each feature is an independent module.

🔗

API-First Backend

REST API designed and tested independently before frontend integration.

🧪

Continuous Testing

Each feature tested manually after implementation across all three user roles.

📐
Design Principles

🎯 User-Centered Design

Separate dashboards tailored to each role's specific needs and workflows

🔒 Security First

JWT auth, bcrypt hashing, document verification, account approval gates

📱 Responsive UI

Mobile-friendly layouts using CSS Grid and Flexbox throughout

⚡ Performance

Lazy loading, optimistic UI updates, minimal re-renders with React hooks
